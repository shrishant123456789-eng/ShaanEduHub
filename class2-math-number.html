3.1 Numbers up to 1000 üíØ

In Class 1, we learned numbers up to 100. Now, we will go big! We will learn numbers up to 1000.

When we count past 99, we get 100 (One Hundred).

99 + 1 = 100 (Smallest 3-digit number)
999 (Largest 3-digit number)

We read them like this:

  • 101 = One hundred one
  • 150 = One hundred fifty
  • 500 = Five hundred

3.2 Reading and Writing Numbers ‚úçÔ∏è

We can write numbers in two ways: in Figures (digits) and in Words.

Number (Figure) Number Name (Words)
245 Two hundred forty-five
608 Six hundred eight
990 Nine hundred ninety

üí° Tip: Don't forget the zero! For 608, we don't say "sixty eight". We say "Six hundred eight".

3.3 Place Value (H T O) üè†

Every digit in a number has a home. We call this Place Value.

  • H = Hundreds
  • T = Tens
  • O = Ones

Example: 352

3 is in Hundreds place (Value = 300)
5 is in Tens place (Value = 50)
2 is in Ones place (Value = 2)

3.4 Expanded Form üéà

Expanded form means stretching the number to show the value of each digit.

Imagine breaking a toy set into pieces to see what is inside.

456 = 400 + 50 + 6
709 = 700 + 0 + 9
123 = 100 + 20 + 3

3.5 Comparing Numbers üêä

We use symbols to see which number is big or small. Imagine a hungry alligator! The alligator always opens its mouth to eat the bigger number.

  • > (Greater Than): 500 > 200 (Mouth eats 500)
  • < (Less Than): 100 < 900 (Mouth eats 900)
  • = (Equal To): 345 = 345 (Both are same)

Rule: First compare Hundreds place. If same, compare Tens. If same, compare Ones.

3.6 Ascending & Descending Order üì∂

Ascending Order (Going Up) üìà

Small to Big. Like climbing up stairs.

Example: 10, 50, 100, 200

Descending Order (Going Down) üìâ

Big to Small. Like coming down stairs.

Example: 200, 100, 50, 10

3.7 Number Patterns & Skip Counting ‚è≠Ô∏è

Skip counting is like jumping numbers!

Skip Count by 2s:

2, 4, 6, 8, 10... (We skip 1 number in between)

Skip Count by 5s:

5, 10, 15, 20, 25... (Numbers end in 0 or 5)

Skip Count by 10s:

10, 20, 30, 40, 50... (Numbers end in 0)
